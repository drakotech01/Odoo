Attendance Location Validator

Secure and reliable location-based attendance tracking for Odoo

Module Overview

The Attendance Location Validator module extends Odoo's native attendance management by adding robust GPS-based location verification. This feature ensures that employees can only check in and check out within predefined geographic boundaries, effectively minimizing fraudulent attendance and boosting workforce accountability.

Key Features & Benefits

  • Real-time employee location tracking during attendance actions via GPS.
  • Geofence validation against company-specific latitude and longitude coordinates.
  • Prevents attendance recording outside authorized locations with user-friendly alerts.
  • Seamless integration with the Odoo Attendance app for smooth workflow.
  • Easy to configure and deploy with minimal technical setup.

How to Use / Workflow

  1. Install the Attendance Location Validator module from the Odoo Apps Store.
  2. Go to your Company settings and configure the attendance latitude and longitude coordinates.
  3. Use the provided interface to update and save your company's official attendance geofence.
  4. When employees check in or out, their device's GPS location is automatically validated.
  5. Attendance records are created only if employees are within the configured location radius.
